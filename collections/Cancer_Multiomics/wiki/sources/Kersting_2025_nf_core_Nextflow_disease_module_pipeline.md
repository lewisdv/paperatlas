## Notion Summary

**한 줄 요약:** nf-core Nextflow 파이프라인으로 disease module discovery + 평가 + 약물 우선순위 자동화. 6개 알고리즘(DOMINO/DIAMOnD/ROBUST/RWR 등) + 자동 평가(Jaccard/DIGEST/leave-one-out) + 약물 우선순위. inferred module은 알고리즘·입력 네트워크·seed에 따라 크게 달라짐.

**과제 관련성:**

한미암 driver/proteogenomics seed set에 재현 가능한 nf-core 워크플로 적용. 단일 알고리즘이 아닌 다중 메트릭 조합 + Jaccard overlap + DIGEST + leave-one-out 함께 평가 권장. 입력 PPI 네트워크 선택 의존성 + 한 알고리즘 모듈 inflation 위험.
