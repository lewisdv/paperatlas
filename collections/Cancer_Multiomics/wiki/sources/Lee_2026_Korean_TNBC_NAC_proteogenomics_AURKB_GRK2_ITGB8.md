## Notion Summary

**한 줄 요약:** 한국인 stage II–III TNBC 50명 anthracycline+taxane NAC 코호트 baseline n=41 + paired post n=22에서 WES+RNA-seq+TMT proteome+phospho 통합. 5개 저항인자 logistic regression non-pCR AUC 0.946 (Lehmann 0.781). AURKB+GRK2 in vitro druggable, ITGB8/7p21 ADC 후보.

**과제 관련성:**

한국인 TNBC proteogenomic reference (Severance n=50, PRJNA1422845/1422844 + PDC000695/696) — 한미암 한국인 NAC 환자 baseline biopsy 동일 modality 스택 외부 검증. anthracycline+taxane regimen 매칭. AURKB(barasertib)+paclitaxel residual disease 적응증, GRK2(βARK)+paclitaxel Bliss 8.08, ITGB8 ADC stratification 입력값으로 한미암 파이프라인 grounding 가능.

**주요 결과:**

- 코호트: 50명 stage II–III, pCR 38% (19/50), median F/U 33.2개월.
- NMF 5-subtype: immune-enriched pCR 55.6%; xenobiotic 0%; EMT 0%.
- 저항인자: estrogen GSVA / GRK2 PTM-SEA / ITGB8 no-loss / AKR1C2 / ABCA13.
- AUC 0.946 (Lehmann 0.781 / +ESTIMATE 0.818).
- βARK1+paclitaxel Bliss 8.08 (MDA-MB-231); barasertib 10nM + paclitaxel MDA-MB-468/HCC1937/organoid SBO-72 synergy.
- ITGB8 deep loss pCR 80% / shallow loss 41.7% / diploid 12.6% / shallow gain 0% (P=0.036).
