# Calcium imaging readouts

## Current role in this corpus

Calcium imaging is the recurring dish-level neural activity readout that sits alongside MEA when brain organoids are chosen for functional compatibility rather than only lineage identity.

## Working synthesis

- In this collection, calcium imaging is not just a convenient visualization layer. It is one of the readouts that changes which brain protocol family is worth choosing at all.
- Fitzgerald is the main anchor because it treats oscillations, calcium imaging, and MEA as co-designed outputs of a semi-guided cortical workflow.
- Calcium imaging therefore marks the branch where brain organoids are optimized for interpretable dish function before anyone escalates into host validation.

## Main tension

- richer dish-level temporal function versus extra preparation, imaging, and analysis burden
- readout-friendly cortical standardization versus broader developmental diversity
